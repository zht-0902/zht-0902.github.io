js数组常用方法

发表于 2021-08-29 更新于 2022-05-24 分类于 JavaScript
本文字数： 2.7k 阅读时长 ≈ 2 分钟

1.pop方法，删除数组 索引值最大的元素 ，返回值为被删除的那个元素，会影响原数组。

2.push方法，在数组最后添加一个或多个元素，返回值为数组的新长度，会影响原数组。

3.unshift方法，在数组前面插入一个或者多个元素，返回数组的新长度，会影响原数组。

4.shift方法，删除数组第一个元素，返回被删除的那个元素，会影响原数组。

5.reverse方法，将数组内容倒序，返回新的数组，并且会影响原数组

6.join方法，将数组以join指定的字符连接成字符串，返回新数组，不会影响原数组。

7.sort方法， 将数组进行排序(升序),返回新数组，影响原数组

8.splice方法, 在任意位置添加或删除元素,返回删除或被替换的值，如果没有被删除或替换则返回空数组; ,影响原数组

1.只有一个参数时, 从当前位置删除到数组末尾

2.有两个参数时,第一个值为删除的位置，第二个值为删除的个数;

3.有三个或者多个值时，第一个值为插入元素的位置，第二个值为替换的个数，后面的值都为插入的新元素；

9.slice方法

10.map方法

参数1：当前索引值

参数2：当前下标

参数3：原数组

当数组中元素是值类型，map不会改变原数组；当是引用类型，则可以改变原数组

11.indexOf方法，查找某个元素的索引值，如果有多个相同的，则返回第一个值得索引，如果没找到，返回-1

12.find方法，找到第一个满足条件得值，没找到返回undifine

13.forEach方法，数组遍历，没有return，即使有return，也会返回undifine。

14.filter方法，过滤数组
