一、java String类

​ 字符串广泛应用 在 Java 编程中，在 Java 中字符串属于对象，Java 提供了 String 类来创建和操作字符串。

​ 注意: String 类是不可改变的，所以你一旦创建了 String 对象，那它的值就无法改变了析如果需要对字符串做很多修改，那么应该选择使用 StringBuffer&StringBuilder类
